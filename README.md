## Installation

To get started with SimpleCart, follow these simple steps:

-   Clone or download this repo and place it into your server.
-   `composer install `
-   `cp .env.example .env `
-   Create database and modify .env with your DB name and Pusher credentials and Boardcasts.
-   `php artisan migrate`
-   `php artisan serve`
